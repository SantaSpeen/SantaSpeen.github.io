    Anixart Auth methods

    Класс ещё не описан!

    Оглавление класса

    • /auth/signUp, Для регистрации аккаунта.
    • /auth/verify, Для подтверждения регистрации аккаунта.
    • /auth/signIn, Для входа в аккаунт.
    • /auth/firebase, Для <_err_found_data>

    Auth

    /auth/signUp

    Метод для регистрации аккаунта.
    После использования этого метода нужно подтвердить почту методом /auth/verify.

    • Обращение:
      • Протокол: POST
      • Необходимо:
        • Data - Данные запроса:
          • login: str
          • password: str
          • email: str
      • Пример: POST {API_URL}/auth/signUp}
    • Ответ:
      • Тип: json
      • Res: 
        {
  "code": 0, 
  "hash": "hash string", 
  "codeTimestampExpires": 1642752174
}

    /auth/verify

        • Data - Данные запроса:
          • code: int, код, поступивший на почту.
          • login: str
          • password: str
          • email: str
          • hash: str, передаётся в ответе от /auth/signUp

    /auth/firebase

    Метод для <_err_found_data>

    • Обращение:
      • Протокол: POST
      • Необходимо:
        • Data - Данные запроса:
          • token: str
      • Пример: POST {API_URL}/auth/firebase}
    • Ответ:
      • Тип: json
      • Res: 
        {
  "code": 0, 
  "topicName": "<_err_found_data> string"
}
